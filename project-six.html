Merle: The Kinda Incompetent Wizard

Engine: Frostheim
Language: C++
Duration: 15 weeks

Summary

This is our second game created with our own engine. We had a lot of freedom for this project as we were tasked to create a third person game. We chose A Hat In Time as our reference since we enjoyed the player movement and freedom of the game. For this project I continued working with AI and this time we use a behaviour tree. I have also continued working on our animations and added some more functionality.
